Release 6.7 To support QuickFolders, please buy a QuickFolders Pro license here. Or get the new cheaper QuickFolders Standard License. This is the preferred way of funding all the ongoing work and improvements. Read the new comparison chart to see which functions you get depending on your license. You can also donate from here. Improvements
  • New QuickFolders Pro feature: find related mails through pattern matching the currently selected mail. [issue 488] This works via a pattern search which is applied to fields of the currently selected email via a search mask (regular expression).
  • Added an option to hide the 'skip unread folder' button from the Navigation Toolbar.
  • Improved tooltips in settings window.
Bug Fixes
  • Fixed: in Thunderbird 128 the notification popup for restricted features leads to an exception and may lead to failure of some features on systems without a Pro license. [issue 486]
  • Fixed: Keyboard shortcuts (such as ALT+Left ALT+Right) stopped working in Thunderbird 128. [issue 491]
Known Issues
  • Currently the verification of QuickFolders licenses fails when the license is bound to an exchange account when it is accessed using Owl. This is due to a bug in the extensions.accounts Webextensions API - which doesn't include them when using accounts.list() - which we hope will be fixed very soon. See: [issue 482] For up to date news on this, please see the Related bug on Thunderbird's Bugzilla
