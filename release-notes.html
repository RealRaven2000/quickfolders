Known Issues
  • Currently the verification of QuickFolders licenses fails when the license is bound to an exchange account when it is accessed using Owl. This is due to a bug in the extensions.accounts Webextensions API - which doesn't include them when using accounts.list() - which we hope will be fixed very soon.
